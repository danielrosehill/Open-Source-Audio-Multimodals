# Tooling

Tools and libraries for working with audio/multimodal data pipelines.

---

## DataChain

Data pipeline framework for unstructured data (audio, video, images). Useful for building multimodal training datasets.

- **Repository:** [datachain-ai/datachain](https://github.com/datachain-ai/datachain)

---

## Lhotse

Speech data handling library for creating training data pipelines. Integrates with PyTorch and supports audio augmentation, feature extraction, and dataset management.

- **Repository:** [lhotse-speech/lhotse](https://github.com/lhotse-speech/lhotse)

---

## vLLM Omni

Blog post on vLLM's support for omni-modal models including audio.

- **Blog Post:** [vLLM Omni (Nov 2025)](https://blog.vllm.ai/2025/11/30/vllm-omni.html)
