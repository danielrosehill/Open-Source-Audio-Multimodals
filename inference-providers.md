# Inference Providers

Platforms and services offering inference for audio multimodal models.

---

## Dataloop

Model hosting platform with Qwen Audio support.

- **Qwen Audio:** [dataloop.ai/library/model/qwen_qwen-audio](https://dataloop.ai/library/model/qwen_qwen-audio/)

---

## Hugging Face Inference

Hugging Face's inference API for audio-text-to-text models.

- **Models:** [huggingface.co/models?pipeline_tag=audio-text-to-text](https://huggingface.co/models?pipeline_tag=audio-text-to-text&sort=trending)

---

## Replicate

Cloud inference platform with audio model support.

- **Website:** [replicate.com](https://replicate.com/)
